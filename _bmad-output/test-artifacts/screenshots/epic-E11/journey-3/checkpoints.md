# Visual Checkpoints — Journey 3: Create Task Form Validation

## Checkpoint 1: Tasks page loaded
- When: After navigating to /tasks and page is ready
- Screenshot: step-1-tasks-page-loaded.png
- What to look for: My Tasks page with h1 title, Create Task button, status tabs, #f4f2ff background

## Checkpoint 2: Create Task dialog open (empty)
- When: After clicking Create Task button, before filling anything
- Screenshot: step-2-dialog-open-empty.png
- What to look for: Dialog visible with Title field (required), Description textarea, Priority select, Due Date picker, action buttons

## Checkpoint 3: Validation error after empty submit
- When: After clicking Create button without filling the title field
- Screenshot: step-3-validation-error.png
- What to look for: Red error text below Title field indicating title is required, dialog still open, Create button not in loading state
