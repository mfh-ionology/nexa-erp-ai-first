# Journey 19: Dashboard Task Status Toggle — Visual Checkpoints

## Checkpoint 1: Dashboard with Tasks Today card showing active tasks
- **When**: After navigating to dashboard and scrolling to bottom cards
- **Screenshot file**: step-1-dashboard-tasks-today-with-tasks.png
- **What to look for**: Tasks Today card visible with CheckSquare icon, active tasks listed with grey Circle status icons (OPEN) or blue CircleDot (IN_PROGRESS), task titles, priority badges. Card has purple-tinted shadow on hover.

## Checkpoint 2: After clicking status icon — task cycles to IN_PROGRESS
- **When**: After clicking the status icon of an OPEN task in the Tasks Today card
- **Screenshot file**: step-2-status-cycled-to-in-progress.png
- **What to look for**: The clicked task's status icon changed from grey Circle to blue CircleDot. The task remains in the card (still active). The change is reflected immediately without page reload.
