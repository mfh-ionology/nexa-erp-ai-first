# Visual Checkpoints — Journey 5: Open Task Detail Sheet

## Checkpoint 1: Task Detail Sheet Open
- **When**: After clicking the first task row (step 2)
- **Screenshot file**: step-2-detail-sheet-open.png
- **What to look for**: 480px Sheet visible from right side. Header shows "Task Details" in serif font. Task title displayed prominently with pencil edit icon. Status section with action buttons (Start/Complete/Cancel for OPEN tasks). Description section visible. Priority Select dropdown. Due Date with calendar picker. Assignees list with avatars. Linked Record section if entity linked. Activity timeline at bottom. Footer with red "Delete Task" button.

## Checkpoint 2: Activity Timeline Section
- **When**: After verifying activity timeline (step 3)
- **Screenshot file**: step-3-activity-timeline.png
- **What to look for**: Activity timeline section visible in the detail sheet. At least one "Task created" entry with date. Purple dot indicators connected by vertical line. Timeline entries show timestamps.
