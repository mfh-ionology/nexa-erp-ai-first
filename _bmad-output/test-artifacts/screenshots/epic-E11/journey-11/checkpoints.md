# Journey 11: Batch Selection and Actions — Visual Checkpoints

## Checkpoint 1: Tasks page loaded with table

## Checkpoint 2: Two tasks selected, batch action bar visible
- **When**: After checking checkboxes on first and second task rows
- **Screenshot**: step-3-two-tasks-selected-batch-bar.png

## Checkpoint 3: After batch Complete All — both tasks completed
- **When**: After clicking 'Complete All' in the batch action bar
- **Screenshot**: step-4-batch-complete-all-done.png
- **What to look for**: Both previously selected tasks now show green CheckCircle2 icons, line-through titles, 'Completed' status labels. Batch action bar dismissed. Selection cleared. Success toast visible
