# Visual Checkpoints — Journey 4: Verify Template List Grouping and Badges

## Checkpoint 1: Template List Page Loaded with Grouped Sections
- **When**: After step 1 — navigating to /settings/document-templates
- **Screenshot file**: step-1-template-list-grouped.png
- **What to look for**:
  - Page title "Document Templates" visible
  - Multiple accordion sections visible, grouped by document type (e.g. "Sales Invoice", "Delivery Note")
  - Each accordion header shows a purple badge with the document type label and a template count
  - Filter bar visible at top with "All Document Types" dropdown, search input, and "Show Inactive" button
  - "Add Template" purple button visible in action bar

## Checkpoint 2: Default Invoice Template Card with Badges
- **When**: After step 3 — verifying the default template card in SALES_INVOICE section
- **Screenshot file**: step-3-default-invoice-card-badges.png
- **What to look for**:
  - Template card with "Standard Invoice" name clearly visible (12px rounded corners)
  - Green "Active" badge visible on the card
  - Purple "Default" badge visible next to Active badge
  - Version count displayed (e.g. "0 versions")
  - Page size metadata "A4 / portrait" visible
  - Three-dot overflow menu button visible on card

## Checkpoint 3: DELIVERY_NOTE Section Collapsed
- **When**: After step 7 — clicking to collapse the DELIVERY_NOTE accordion section
- **Screenshot file**: step-7-delivery-note-collapsed.png
- **What to look for**:
  - DELIVERY_NOTE accordion header visible but section content hidden
  - Only the header row showing "Delivery Note" label and count badge
  - Other expanded sections still visible above/below
  - Chevron icon pointing in collapsed direction
