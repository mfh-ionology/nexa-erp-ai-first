# Visual Checkpoints — Journey 13: Corrections Tab — Filtering by Type, Skill, Date, Resolved Status

## Checkpoint 1: Corrections Tab Initial Load
- **When**: After navigating to /ai/admin/knowledge#corrections (Step 1)
- **Screenshot**: `step-1-corrections-tab-loaded.png`
- **What to look for**: Corrections tab is active, filter bar visible with type filter, skill key input, auto-resolved toggle (All/Resolved/Pending), date range inputs. Stats cards showing correction totals. Accordion groups of corrections visible below filters.

## Checkpoint 2: After TERMINOLOGY Type Filter Applied
- **When**: After selecting TERMINOLOGY from the type filter (Step 2)
- **Screenshot**: `step-2-terminology-filter-active.png`
- **What to look for**: Type filter button shows "Terminology" or "1 type" with purple active styling. Correction list filtered to only show TERMINOLOGY-type corrections.

## Checkpoint 3: After Skill Key Filter Applied
- **When**: After entering "create_invoice" in the skill key filter (Step 3)
- **Screenshot**: `step-3-skill-key-filtered.png`
- **What to look for**: Skill key input contains "create_invoice" with purple active border. Corrections further filtered.

## Checkpoint 4: After Auto-Resolved Toggle Set to Resolved
- **When**: After clicking "Resolved" toggle button (Step 4)
- **Screenshot**: `step-4-resolved-only.png`
- **What to look for**: "Resolved" button highlighted with active styling. Only auto-resolved corrections shown.

## Checkpoint 5: After Date Range Applied (All Filters Active)
- **When**: After setting date range 2026-02-01 to 2026-03-04 (Step 5)
- **Screenshot**: `step-5-all-filters-active.png`
- **What to look for**: All filter controls show active state (purple borders/backgrounds). Date inputs populated. Clear button visible. Filtered corrections list reflects all combined filters.

## Checkpoint 6: After Clearing All Filters
- **When**: After clicking Clear filters button (Step 6)
- **Screenshot**: `step-6-filters-cleared.png`
- **What to look for**: All filters reset to defaults — type shows "All Types", skill key empty, auto-resolved toggle on "All", date inputs empty. Clear button hidden. Full unfiltered corrections list restored.
