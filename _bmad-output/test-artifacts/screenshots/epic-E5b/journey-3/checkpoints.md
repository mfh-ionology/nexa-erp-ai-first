# Visual Checkpoint Manifest — Journey 3: Toggle Memory Settings

## Checkpoint 1: Memory page loaded with settings panel
- **When**: After navigating to /ai/memory and page loads
- **Screenshot file**: step-1-memory-page-loaded.png
- **What to look for**: Memory page with Concept D styling: light purple (#f4f2ff) background, settings panel visible with "Enable AI Memory" toggle in ON state (purple), category checkboxes all checked, retention period selector visible

## Checkpoint 2: Memory toggle switched OFF — settings disabled
- **When**: After clicking the "Enable AI Memory" toggle to OFF
- **Screenshot file**: step-2-toggle-off-settings-disabled.png
- **What to look for**: Toggle switch is in OFF state (grey, not purple), category checkboxes are visually disabled/greyed out, retention period selector is disabled. The memory list below should still be visible but the settings controls are inactive.

## Checkpoint 3: Retention period changed to 90 days
- **When**: After selecting "90 days" in the retention period selector
- **Screenshot file**: step-6-retention-90-days.png
- **What to look for**: Retention period selector displays "90 days" as the selected value. Toggle should be back ON (purple), all category checkboxes should be checked and enabled.
