# Phase 1.2a Governance Baseline (Official)

Status: OFFICIAL baseline for all future Nexa ERP build work.

Scope and sources:
- Master Plan (single source of truth): `docs/master-plan/nexa-erp-v1-master-plan.md`
- Gap Matrix (evidence of current state vs plan): `reports/verification/phase-1.2a-gap-matrix-*/gap-matrix.*`
- Execution Plan: `docs/plans/phase-1.2a-execution-plan.md`
- Approval Record: `docs/approvals/phase-1.2a.md`

Policy:
- Phase 1.2a Option A is governance-only: enumerate, verify, and plan. It is not full implementation.
- All future phases must reference the Master Plan and the latest approved Gap Matrix, Execution Plan, and Approval Record.
- Change-control: no scope changes without updating the Master Plan (explicitly), regenerating the Gap Matrix, updating the Execution Plan, and re-approving in `docs/approvals/phase-1.2a.md`.

Blocking rule:
- No implementation beyond this baseline may proceed without an approved plan and approval record as specified above.
