# Manifest Minimums (Phase 1.1b1 — Finance)

- Finance minimums (from plan): entities ≥ 10, flows ≥ 10, invariants ≥ 8, propagation ≥ 6.
- All items are required to be `status: planned` with a valid `plan_ref`; `planned-but-undefined` fails coverage.
- Plan references must point to sections extracted in `plan-outline.json` (generated in Phase 1.1a).

Modules beyond Finance will be added in later phases with their own minimums.
