## Dev Notes

### CRITICAL: What This Story IS and IS NOT

**IS:**
- A visual/UI layer replacement — swapping component implementations and CSS
- Adapting v0's Next.js code to our Vite + TanStack Router stack
- Converting hardcoded strings to i18n translation keys
- Adding Recharts for interactive charts (replacing raw SVG)

**IS NOT:**
- A backend change (no API, no Prisma, no event bus changes)
- A new feature (all functionality already exists or is mock data)
- A state management change (keep Zustand + TanStack Query)
- A routing change (keep TanStack Router file-based routes)

### Framework Adaptation Cheatsheet (Next.js → Vite + TanStack Router)

| Next.js | Nexa Equivalent | Notes |
|---------|-----------------|-------|
| `import Link from "next/link"` | `import { Link } from '@tanstack/react-router'` | `href` → `to` prop |
| `usePathname()` | `useRouterState({ select: s => s.location.pathname })` | Or `useMatches()` |
| `"use client"` | Remove entirely | Vite has no RSC |
| `export default function Page()` | `export const Route = createFileRoute(...)({ component: Page })` | TanStack Router convention |
| `import { useRouter } from "next/navigation"` | `import { useNavigate } from '@tanstack/react-router'` | `router.push()` → `navigate()` |
| `<style jsx>` | Move to `globals.css` | Vite doesn't support styled-jsx |

### Previous Story Intelligence (E6-14)

**Key Learnings to Apply:**
1. **i18n keys must be flat** — `keySeparator: false` means `t('dashboard.greeting.morning')` is a flat key lookup, NOT nested
2. **No `||` fallbacks** for missing i18n keys — `t()` returns key name as truthy string, making fallbacks dead code
3. **`common.json` must be valid JSON** — trailing commas before closing braces cause build failures
4. **SVG colors use CSS custom properties** — `var(--color-primary)` not hardcoded hex
5. **Checkbox `disabled` prop** needed for non-interactive checkboxes
6. **All interactive elements** need proper ARIA attributes
7. **Tailwind `@theme` block** is the SOLE source of truth — no duplicate HSL variables
