## Dev Notes

### Architecture Context

This story delivers the **Platform Admin App Shell** — a separate React SPA (`apps/platform-admin/`) for vendor super-administrators. It is completely independent from the tenant ERP web app (`apps/web/`). Key distinctions:

| Aspect | ERP Web App (`apps/web/`) | Platform Admin (`apps/platform-admin/`) |
|--------|--------------------------|----------------------------------------|
| Users | Tenant employees | Vendor operators |
| Auth | ERP JWT + company context | Platform JWT + MFA (BR-PLT-018) |
| Database | Tenant PostgreSQL | Platform PostgreSQL |
| API target | `apps/api/` (port 5100) | `apps/platform-api/` (port 5101) |
| Dev port | 5110 | 5112 |
| Theme | Light (Concept D purple) | Dark sidebar (slate-900 + purple accent) |
| RBAC | Access Groups with resource-level perms | Two roles: PLATFORM_ADMIN, PLATFORM_VIEWER |
